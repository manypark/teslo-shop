# Teslo API

1. Clonar proyecto
2. ```npm i ```
3. Clonar el archivo de ```.env.template```y renombralo a ```.env```
4. Cambiar las vairables de entorno
5. Levantar la db
```
docker-compose up
```
6. Levantar server: ```npm start:dev```
